> The we can get the first element of a list using
> ```py
> first = arr[0]
> ```

> The we can get the last element of a list with negative indexing. Negative index starts at -1 as opposed to the positive index.
> ```py
> last = arr[-1]
> ```

> Expressions such as `a > b` evaluate to either `True` or `False` 
> ```py
> a = 3
> b = 1
> is_true = a > b # is_true would be True here 
> ```
